# pa-zhihu

## 环境

 - python 3.3以上
 - gevent
 - lxml
 - requests
 - pymysql
 - pymongo
 - 登陆界面，是模仿的7sdream的[zhihu-py3](https://github.com/7sDream/zhihu-py3)。
